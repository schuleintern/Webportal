◆ $maxResourceSize

Sabre\CardDAV\Plugin::$maxResourceSize = 10000000
protected

The default PDO storage uses a MySQL MEDIUMBLOB for iCalendar data, which can hold up to 2^24 = 16777216 bytes. This is plenty. We're capping it to 10M here.
