Ausführliche Beschreibung

Stellt allgemein einige Datumsfunktionen zur Verfügung. z.B. kann mit dieser Klasse mit natürlichenn Datumsangaben (TT.MM.JJJJ) umgegangen werden Außerdem können SQL Datumsangaben verwndet werden. (YYYY-MM-DD)
