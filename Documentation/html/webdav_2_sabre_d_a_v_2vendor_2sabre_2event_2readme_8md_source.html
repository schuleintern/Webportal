1 sabre/event
2 ===========
3 
4 A lightweight library for event-based development in PHP.
5 
6 This library provides the following event-based concepts:
7 
8 1. EventEmitter.
9 2. Promises.
10 3. An event loop.
11 4. Co-routines.
12 
13 Full documentation can be found on [the website][1].
14 
15 Installation
16 ------------
17 
18 Make sure you have [composer][3] installed, and then run:
19 
20  composer require sabre/event "~3.0.0"
21 
22 This package requires PHP 5.5. The 2.0 branch is still maintained as well, and
23 supports PHP 5.4.
